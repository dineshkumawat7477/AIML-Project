Project Title
Expense Tracker

Overview of the Project
Expense Tracker is a GUI-based application written in Python using Tkinter, designed to help users manage and monitor their daily expenses efficiently. The application allows users to categorize expenses, add details, and keep a monthly summary.

Features
User-friendly graphical interface
Categorize expenses (Food, Laundry, Person, Other, Credit)
Enter custom details and amounts
Data stored in a local file for persistence
Monthly and daily sorting with view functionality
Technologies/Tools Used
Python 3.13
Tkinter library (for GUI)
Standard libraries: os, time
Steps to Install & Run the Project
Make sure Python 3.x is installed on your system.
Clone the repository or download all project files in one folder.
Ensure that a file named Storage.txt exists in the directory (or it will be created automatically).
Run the main file using the command: python main.py (in Windows) python3 main.py (in Linux or Mac OS)
